## Features

✅ **Offline-First** - All data stored locally by default  
✅ **Optional Cloud Sync** - Sign in to backup and sync across devices  
✅ **No Ads** - Just your movies, your browser  
✅ **Private** - You own your data  
✅ **Import/Export** - Full data portability  

---

## Quick Start

### Offline Mode (No Account)
1. Open Log!t
2. Click "Continue Offline"
3. Add and rate movies
4. Everything stays on your device

### Cloud Mode (Optional)
1. Open Log!t
2. Click "Sign in with Google" (or GitHub)
3. Existing movies upload automatically
4. Changes sync across devices

---

## Cloud Features (v3.0.0+)

- **Authentication** - Google, GitHub, Email
- **Cloud Sync** - Automatic backup of your library
- **Multi-Device** - Access your movies anywhere
- **Offline Fallback** - Works great without internet
- **Data Control** - Export/import anytime, delete account anytime

### Setup Cloud Features

Cloud features are **optional**. To enable:

1. See [Cloud Setup Guide](CLOUD_SETUP.md) for full instructions
2. Or run the quick setup:
   - Open `/config.html`
   - Add your Supabase project credentials
   - Done!
